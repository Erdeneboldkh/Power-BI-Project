# Power-BI-Project
Salary & Work-Life Balance Happiness Dashboard

This Power BI project analyzes how salary, work-life balance, and job title affect happiness across different countries.

It explores which countries and job titles offer the best combination of income, balance, and overall happiness.

Skills Demonstrated

Data cleaning and transformation in Power Query

Data modeling & creating relationships

Creating calculated columns and DAX measures

Building KPIs & interactive slicers

Designing clear and actionable dashboards
